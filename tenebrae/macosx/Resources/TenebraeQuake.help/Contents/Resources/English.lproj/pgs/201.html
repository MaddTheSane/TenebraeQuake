Using mission packs by dragging them onto the TenebraeQuake application icon

You can use mission packs by simply dragging them onto the TenebraeQuake application icon.

Please follow this steps to use mission packs or mods via drag and drop:

  1. Copy the folder of the mission pack to your "Quake" folder. The "Quake" folder is the folder that holds the "id1" folder.

  2. Drag the folder which contains the mission pack onto the TenebraeQuake application icon.
