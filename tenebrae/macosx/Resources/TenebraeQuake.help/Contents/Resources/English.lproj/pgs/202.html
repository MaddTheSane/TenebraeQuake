Using mission packs from the startup dialog

You can select mission packs or mods in the "CLI" panel of the startup dialog.

Please follow this steps to use official mission packs from the startup dialog:

  1. Hold the "Option" key while starting the TenebraeQuake application.

  2. Go to the "CLI" settings panel.

  3. Select the + button and select either "-hypnotic" or "-rogue" in the pop up menu.

  4. Press the OK button.

Please follow this steps to use mods from the startup dialog:

  1. Hold the "Option" key while starting the TenebraeQuake application.

  2. Go to the "CLI" settings panel.

  3. Select the + button.

  4. Select the "-game" option in the pop up menu.

  5. Enter the name of the mod in the text field.

  6. Press the OK button.
