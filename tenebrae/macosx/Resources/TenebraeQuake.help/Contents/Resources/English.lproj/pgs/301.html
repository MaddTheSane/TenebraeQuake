About "vispatch"

The application "Vispatch" is a utility that patches your Quake levels to support transparent water with TenebraeQuake (and GLQuake/GLQuakeWorld).

The "Vispatch" application is a command line application, which has to be launched from the "Terminal". The application can be found inside the "Extras" folder that comes with this version of TenebraeQuake.
