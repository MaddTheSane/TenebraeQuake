Obtaining "vis" patches

You can download "vis" patches for your favorite mission packs or mods at:

"http://http://www.atomicgamer.com/directory.php?id=3139"

The files are usually packed with "zip" and contain several files. You'll need only one file from such a downloaded archive: The file that has the ".vis" extension. Say you have downloaded the archive "hipnotic.zip" for use with the Hipnotic mission pack. All you need for use with "vispatch" is the file called "hipnotic.vis".
