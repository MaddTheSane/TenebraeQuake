Executing TenebraeQuake console commands with AppleScript

You can execute Quake console commands with the "console" command while TenebraeQuake is running. The syntax for the "console" command is as follows:

console execute String

The following example will tell the appication "TenebraeQuake" to open the map "e1m8":

tell application "TenebraeQuake"
    console execute "map e1m8"
end tell

You have to separate each command with a semicolon if you want to execute several Quake console commands with a single "console" line. For example:

tell application "TenebraeQuake"
    console execute "toggleconsole;quit"
end tell
