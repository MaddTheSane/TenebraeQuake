Mouse movement is too slow

If the maximum mouse speed at the options dialog doesn't fit your needs, you should go to the TenebraeQuake console and experiment with the variable "sensitivity". Entering "sensitivity" without any parameter will report the current mouse sensitivity value. Entering "sensitivity x", for example "sensitivity 44", will set the mouse speed to the value of x.
