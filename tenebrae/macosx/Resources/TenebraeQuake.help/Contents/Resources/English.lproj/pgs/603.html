I get an invalid video mode with TenebraeQuake

If you get a video mode, which your monitor can't display upon the first start of TenebraeQuake, you should do the following:

Please follow this steps:

  1. Open "TextEdit" which comes with the MacOS X installation.

  2. Locate the file "config.cfg" inside your "id1" folder and open it with "TextEdit".

  3. Edit the line "_vid_default_mode x" within the "id1/config.cfg" file and change the value of "x" until you get a working display mode. For example: "_vid_default_mode 0". Mode "0", "1" and "2" are windowed modes.
