I want to use TenebraeQuake but I'm told that my Mac has too less memory

If you have used the "Classic RAVE" or the "PC" version of Quake and want to use TenebraeQuake, you have to delete the "glquake" folder inside the "id1" folder:

Please follow this steps:

  1. Go to the Finder.

  2. Locate the "glquake" folder inside your "id1" folder.

  3. Drag the "glquake" folder to the "Trash".
