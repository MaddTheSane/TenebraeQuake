Game Pads and Rumble

TenebraeQuake supports rumble with any game pad that comes with a force feedback driver. Below is a list of game pads that have been tested.

Supported game pads:

  • Microsoft Xbox 360 Controller (requires driver▸).

Unsupported game pads:

  • Sony DUALSHOCK®2 for PlayStation®2 (with Play.com Dual PS2 to PC Convertor).

  • Sony DUALSHOCK®3 for PlayStation®3.

You can enable or disable the rumble effect from the TenebraeQuake options menu. It is disabled by default.
