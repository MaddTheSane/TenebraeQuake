Adjusting the rumble duration

The rumble effect is triggered when the player is receiving damage. You can adjust the duration of the rumble effect from the TenebraeQuake console via the variable:

actuatorduration seconds

The duration is set by default to 0.2 seconds. To change the duration to 0.5 seconds you would enter:

actuatorduration 0.5
