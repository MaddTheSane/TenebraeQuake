Installing TenebraeQuake

The MacOS X port of TenebraeQuake requires the data files of the retail or shareware version of Quake.

Please follow this steps to install the MacOS X version of TenebraeQuake:

  1. Install the Classic MacOS or PC version. Only the "id1" folder is required.

  2. Delete the "glquake" folder, which is located inside the "id1" folder.

  3. Copy the TenebraeQuake application to your Applications folder.

  4. Copy the "tenebrae" folder to your "Quake folder". The "Quake folder" is the folder that holds the "id1" folder.

  5. Optional: TenebraeQuake requires "watervised" maps. Please follow instructions on how to patch your existing PAK files at the "Transparent Water" section of this help.

Note: You will be asked to locate the "id1" folder which comes with the shareware or retail version of Quake, if you startup TenebraeQuake for the first time and have not placed the TenebraeQuake application inside your Quake folder (the "Quake folder" is the folder that holds the "id1" folder).
