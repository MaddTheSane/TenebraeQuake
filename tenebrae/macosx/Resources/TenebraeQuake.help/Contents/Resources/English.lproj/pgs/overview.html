Overview

This is the Cocoa version of the enhanced Quake client TenebraeQuake. TenebraeQuake adds for example stencil shadows and bump-mapping to Quake.

System Requirements:

  • An Intel Mac.

  • MacOS 10.6 or later.

  • The shareware or retail version of Quake with watervised levels.

The port comes with the following files:

  • TenebraeQuake: The TenebraeQuake application.

  • tenebrae: A folder containing data files required by TenebraeQuake.

  • Extras/vispatch: Required to apply water transparency patches. A command line application.

  • Extras/AppleScript Examples/: This folder contains several example AppleScripts for use with TenebraeQuake.
